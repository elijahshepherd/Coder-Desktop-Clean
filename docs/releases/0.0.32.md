# Coder Desktop - Version 0.0.32

**Release date:** June 6, 2026

**Version:** `0.0.32`

**Status:** Patch

## Overview

Coder Desktop 0.0.32 focuses on safer update behavior, calmer settings, background provider diagnostics, and stricter image generation control.

This release changes update prompts to open the correct GitHub Release asset instead of attempting an in-app install, removes manual provider-test UI, moves theme switching into Settings, limits image generation to NVIDIA FLUX models, and fixes cancellation so stopping a chat does not trigger provider-error retries.

## What changed

- Update cards now show `Install` and `Not Now`.
- Install opens the matching GitHub Release file for the user's operating system and CPU architecture.
- Manual provider test controls were removed from Settings.
- Provider health checks now run automatically in the background and report sanitized evidence only when problems occur.
- Light mode and Dark mode now live in Settings and persist between app launches.
- Visible Coder Desktop logo rendering no longer uses a filled logo background.
- Image generation now uses the duplicated black-and-white spews animation asset.
- Copy image link works for generated images.
- Image generation cards no longer show `Image provider` text in the visible layout.
- Personalization settings keep a cleaner layout with simpler theme controls.
- Image generation models are limited to NVIDIA FLUX models.
- The assistant can request safe settings changes through a controlled settings tool.
- Stopping chat generation now cancels immediately and prevents retries.
- Failed image generation tries configured FLUX fallback models and shows one concise status message.
- Image generation stops after the requested count and supports one to three images in one response.
- Image prompts are guided toward detailed, editable visual instructions.

## Verification

- Targeted Vitest regression tests for updates, provider health checks, settings tools, store sanitization, and image generation.
- Full TypeScript type checking.
- Full Vitest suite.
