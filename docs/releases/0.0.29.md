# Coder Desktop - Version 0.0.29

**Release date:** June 6, 2026

**Version:** `0.0.29`

**Status:** Patch

## Overview

Coder Desktop 0.0.29 focuses on settings and chat UI polish. This release repairs the visual issues introduced around the full-screen settings surface and the mobile chat start page.

The goal is to make settings feel like a stable desktop product surface again: opaque, readable, correctly tabbed, and usable on narrow screens.

## What Changed

- Settings now keeps its desktop tab list in a fixed sidebar layout instead of stretching every tab across the top of the modal.
- Mobile settings now uses a horizontal tab rail with the settings content below it.
- Provider cards now use the intended responsive grid after all legacy settings styles have loaded.
- The settled settings panel now stays opaque so the chat page does not visually bleed through.
- The mobile starter card now keeps a readable width and stacks its action buttons.
- Save settings now appears in the providers heading instead of clipping at the bottom of the modal.
