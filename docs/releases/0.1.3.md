# Fix missing diff dependency and build reliability

Coder Desktop 0.1.3 resolves a missing dependency that caused startup crashes on fresh installs.

---

## Highlights

- Fixed `Cannot find module 'diff'` error on fresh Windows/macOS installs
- Verified production build completes successfully

## Added

- `diff` package now properly bundled in production builds (was in package.json but not installed in CI)

## Other Changes

- Confirmed Windows NSIS installer and macOS ZIPs build and run correctly

---

## Install

**Windows**: Download `Coder-Desktop-0.1.3-setup-win-x64.exe` and run it. The installer supports automatic updates.

**macOS (Intel)**: Download `Coder-Desktop-0.1.3-mac-x64.zip`, extract, and move `Coder Desktop.app` to Applications.

**macOS (Apple Silicon)**: Download `Coder-Desktop-0.1.3-mac-arm64.zip`, extract, and move `Coder Desktop.app` to Applications.

All artifacts include SHA256 checksums in the release manifest.

---

## Notes

- This is a patch release fixing a build/packaging issue only
- No new features or UI changes
- macOS builds must run on macOS (GitHub Actions or local)
- Windows SmartScreen may warn on unsigned builds; configure `CSC_LINK` and `CSC_KEY_PASSWORD` secrets for code signing
