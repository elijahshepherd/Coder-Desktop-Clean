# Summary

Version 0.0.43 overhauls the maintenance system with a blocking overlay, makes the update notification clearer with a mid-screen modal, and polishes the settings layout.

## Maintenance system

- **Blocking card overlay** — When maintenance is active, a full-screen overlay now appears with the ESID, OF, AL, note, and issue number, preventing all interaction until complete
- **Instant push on detection** — Maintenance state is broadcast to the renderer immediately on poll and on startup, not just on the 5-minute interval
- **Resume on close** — The overlay disappears automatically when the maintenance issue is closed

## Update notification redesign

- **Mid-screen modal** — The update notice is now a centered dialog with a dark scrim, replacing the bottom-right corner notification
- **Blocks interaction** — The modal prevents interaction with the app until dismissed via the Done button
- **Progress inside card** — Download progress is shown inside the modal card
- **Clean install flow** — Clear messaging for each phase

## Settings polish

- **Access tab renamed to Security** — The settings tab label now matches the panel heading
- **Gray background removed** — The gray block behind settings tab content has been removed for a cleaner look
- **Warning text spacing** — Added top margin to the auto-accept warning for better readability

## Sub Apps

- **Wide page layout** — The Sub Apps modal now uses a wider layout (720px) with room for more cards
- **More placeholder cards** — Five app cards shown (Image Studio, Sandbox, Web Explorer, Data Viewer, Agent Hub) with coming-soon labels
