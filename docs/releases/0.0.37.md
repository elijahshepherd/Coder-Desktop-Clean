# Coder Desktop - Version 0.0.37

**Release date:** June 7, 2026

**Version:** `0.0.37`

**Status:** Patch

## Overview

Coder Desktop 0.0.37 refines the Personalization settings UI after the annotated review of version 0.0.36.

This release removes decorative selection lines, colored tone dots, the accent preview chip, and round feedback switch markers from the Personalization tab. The controls keep their behavior, but the visible design is quieter and more neutral.

## What changed

- Removed the decorative vertical line from the Button tone card.
- Removed active underline bars from theme and tone buttons.
- Removed colored dots from tone buttons.
- Removed the custom accent color swatch from the picker row.
- Removed round markers from completion feedback switches.
- Kept selected tone, theme, and feedback states visible with neutral borders and surfaces.
- Kept the Personalization layout responsive without horizontal overflow.

## Verification

This release was checked with:

- TypeScript type checking.
- Screenshot-driven UI inspection on the Personalization settings page.
- Interactive tone selection checks, including selecting Green and confirming the panel stayed neutral.
- Mobile layout metrics for the Personalization settings page.
- Horizontal overflow checks.
