# Summary

Version 0.0.42 improves SmartScreen compatibility for Windows users by enhancing executable metadata and adjusting the build configuration.

## Windows SmartScreen improvements

SmartScreen is Microsoft's reputation-based security feature that warns users before running unrecognized executables. Without a code signing certificate, SmartScreen may show a warning for new releases, especially if the executable lacks complete metadata.

This release addresses the metadata gap:

- **LegalTrademarks** added to all Windows executables
- **FileDescription** expanded to include the full product description
- **PrivateBuild** and **SpecialBuild** fields explicitly cleared
- **NSIS installer** metadata enhanced with `VIProductVersion`, `BrandingText`, and all `VIAddVersionKey` fields
- **Installer icons** explicitly set for the installer, uninstaller, and header
- `signAndEditExecutable` and `verifyUpdateCodeSignature` explicitly disabled for unsigned builds

## Build configuration

- NSIS installer is now the primary Windows build target
- Portable and ZIP remain as secondary formats
- Updated the ZIP start guide with "More info → Run anyway" instructions for SmartScreen prompts

## What this changes

The code, features, and behavior are identical to v0.0.41. Only metadata and build configuration changed.
