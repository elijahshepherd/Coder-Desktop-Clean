# Coder Desktop - Version 0.0.31

**Release date:** June 6, 2026

**Version:** `0.0.31`

**Status:** Patch

## Overview

Coder Desktop 0.0.31 refreshes the app identity, improves personalization, and fixes reliability issues reported from real 0.0.30 testing.

This release adds Elijah's new light and dark Coder Desktop logo artwork, stronger continuation-prompt privacy safeguards, safer provider-test behavior, faster selected-provider image generation, personalization polish, and app-wide font consistency.

## What changed

- The app now uses the new logo artwork, with the dark logo in light mode and the light logo in dark mode.
- Windows and macOS package icon files were regenerated from the new artwork.
- Personalization now has calmer active states, completion feedback, and theme motion while keeping the interface readable.
- Hidden continuation recovery text is no longer formatted as a Markdown heading and is stripped before any assistant text can be saved to chat.
- Provider testing skips missing API keys as setup work instead of counting them as failed checks.
- NVIDIA image model 404 responses are treated as account or endpoint availability issues, so known image models are kept instead of being removed incorrectly.
- Provider test reporting avoids duplicate high-severity GitHub issues for expected setup gaps, fallback timeouts, no-text fallback responses, and account-specific NVIDIA image availability checks.
- Image generation always uses the active provider selected in settings, even if an assistant image request names a different provider.
- Image generation now fails faster when the selected provider cannot create the requested image.
- The direct image-request detector now handles typo variants such as `generat an image`.
- Code blocks, command output, percentages, durations, and report details now inherit the same app-wide font.
- Settings and chat surfaces have calmer panel transitions, hover polish, and progress styling.

## Verification
