# New logos with system theme detection

Coder Desktop now uses updated logos that automatically adapt to your system theme.

---

## Highlights

- Light logo on dark mode, dark logo on light mode
- System theme option follows OS preference automatically

## Added

- System theme option ("System") that detects and follows OS dark/light preference
- New light/dark logo variants from updated brand assets

## Fixed

- Logo contrast now correct in both light and dark modes
- App icon uses dark logo as default (works on light taskbars)

## Other Changes

- Updated logo conversion script to use separate light/dark source images
- Default theme changed to "system" for automatic adaptation
