# Coder Desktop - Version 0.0.30

**Release date:** June 6, 2026

**Version:** `0.0.30`

**Status:** Patch

## Overview

Coder Desktop 0.0.30 focuses on update reliability, settings behavior, provider diagnostics, and dark-mode consistency.

This release fixes the update handoff path so the app does not close until the downloaded update is verified and the install helper has started. It also turns settings into an auto-saving surface, adds a full provider test, improves personalization, and makes strict Let Me Know behavior easier to understand and control.

## What changed

- Update installation now verifies the downloaded file before starting the installer or portable replacement helper.
- The app waits for the update helper process to start before quitting, so update errors can stay visible instead of closing the app.
- Dark mode no longer shows the new-version shimmer animation on update notices or update progress bars.
- Settings changes now save immediately, and the manual Save settings button has been removed.
- The AI functionality tab now uses a brain icon.
- Max Let Me Knows is now a switch-style button instead of a slider.
- Strict Let Me Know mode now tells the assistant to ask before inferring missing details that could change the result.
- Personalization now includes a custom accent color with light and dark mode readability safeguards.
- Provider settings now include a Provider test action for base URLs, primary models, fallback models, and supported image models.
- Provider test failures remove clearly broken fallback or image models and can promote a working fallback model.
- Provider test reports send sanitized evidence for developer investigation without API keys.
