# Improved Windows security and uninstall reliability

Coder Desktop 0.1.2 enhances Windows compatibility with proper application manifests, force uninstall recovery, and code signing readiness.

---

## Highlights

- Windows app manifest with PerMonitorV2 DPI awareness and Windows 10/11 compatibility declarations
- Force uninstall via CLI, in-app settings, or NSIS uninstaller checkbox
- Code signing infrastructure ready for trusted certificate deployment

---

## Added

- Windows application manifest (PerMonitorV2 DPI awareness, asInvoker execution level, Win10/11 supportedOS IDs)
- Force uninstall recovery: in-app command, PowerShell CLI script, and NSIS uninstaller checkbox
- CLI uninstall scripts: `npm run uninstall:windows` and `npm run uninstall:windows:userdata`
- Code signing configuration in electron-builder (CSC_LINK, CSC_KEY_PASSWORD environment variables)
- Executable metadata stamping with version, description, and manifest embedding
- AppUserModelId registration on install for proper taskbar integration

---

## Other Changes

- Enhanced NSIS installer with `/FORCE` flag and custom uninstaller page
- Improved executable stamping via rcedit (version strings, comments, manifest)
- Preload API and IPC handler for in-app force uninstall
- Package.json uninstall scripts for easy command-line removal

---

## Install

**Windows**: Download `Coder-Desktop-0.1.2-setup-win-x64.exe` and run it. The installer supports automatic updates.

**macOS (Intel)**: Download `Coder-Desktop-0.1.2-mac-x64.zip`, extract, and move `Coder Desktop.app` to Applications.

**macOS (Apple Silicon)**: Download `Coder-Desktop-0.1.2-mac-arm64.zip`, extract, and move `Coder Desktop.app` to Applications.

All artifacts include SHA256 checksums in the release manifest.

---

## Notes

- macOS application ZIPs must be built on macOS or by the macOS GitHub Actions workflow.
- Windows SmartScreen may warn on unsigned builds; configure `CSC_LINK` and `CSC_KEY_PASSWORD` secrets in GitHub Actions for code signing.
- Force uninstall requires Administrator rights for per-machine installations.
- This is an early release — expect rough edges and rapid iteration.
