# Coder Desktop - Version 0.0.35

**Release date:** June 7, 2026

**Version:** `0.0.35`

**Status:** Patch

## Overview

Coder Desktop 0.0.35 focuses on chat layout reliability, generated-image link reliability, real local path handling, settings polish, and transparent logo packaging.

This release fixes long pasted strings inside chat cards, persists generated images behind short app links, keeps image cards visible while those short links are copied, improves Settings spacing and dark-mode presentation, and rebuilds the desktop icon without an opaque logo background.

## What changed

- Wrapped long one-line chat content, including data URLs, inside the message card without horizontal overflow.
- Added short `coder-image://` generated-image links backed by app-owned image files.
- Reused the same Copy button behavior for generated-image links and normal chat copy actions.
- Swapped image-generation loading to the black-and-white spews animation asset and verified the canvas renders with strong contrast.
- Improved Settings tab spacing, readable text sizing, provider label logos, dark-mode contrast, and mobile layout.
- Kept regular Settings controls icon-free while preserving provider logos on provider labels.
- Rebuilt desktop icon assets with transparent backgrounds.
- Added real user-home path guidance so the assistant does not invent placeholder paths.
- Allowed public repository prompt-location questions to be handled as public source-code questions.
- Recognized `C:/...` Windows paths and treated file paths as files instead of folders.
- Kept cancellation from becoming provider retry behavior or duplicate stopped messages.
