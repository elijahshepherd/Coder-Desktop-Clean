# Updated app logo and icons

Coder Desktop 0.1.5 updates the app branding with new logo variants.

---

## Highlights

- New logo and icon set across the entire application
- Dark mode uses light variant logo, light mode uses dark variant logo
- Taskbar/desktop icon updated to match new branding

## Changed

- Updated `src/renderer/assets/brand/coder-logo-light.png` (light mode logo)
- Updated `src/renderer/assets/brand/coder-logo-dark.png` (dark mode logo)
- Regenerated `build/icon.ico`, `build/icon.icns`, `build/icon.png` from new light mode logo
- Regenerated all renderer logo assets (coder-logo.png, coder-mark.png, coder-logo.svg, coder-mark.svg)

## Other Changes

- Confirmed Windows NSIS installer and macOS ZIPs build and run correctly with new icons

---

## Install

**Windows**: Download `Coder-Desktop-0.1.5-setup-win-x64.exe` and run it. The installer supports automatic updates.

**macOS (Intel)**: Download `Coder-Desktop-0.1.5-mac-x64.zip`, extract, and move `Coder Desktop.app` to Applications.

**macOS (Apple Silicon)**: Download `Coder-Desktop-0.1.5-mac-arm64.zip`, extract, and move `Coder Desktop.app` to Applications.

All artifacts include SHA256 checksums in the release manifest.

---

## Notes

- This is a branding/patch release only
- No new features or UI changes beyond logo updates
- macOS builds must run on macOS (GitHub Actions or local)
- Windows SmartScreen may warn on unsigned builds; configure `CSC_LINK` and `CSC_KEY_PASSWORD` secrets for code signing
