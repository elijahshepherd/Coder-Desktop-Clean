# Updated app logo and brand assets

Coder Desktop 0.1.1 refreshes the application logo and brand identity across all platforms.

---

## Highlights

- New Coder Desktop logo with improved visibility in both light and dark modes
- Updated icons for Windows (ICO), macOS (ICNS), and Linux (PNG)

## Added

- Inverted logo variant (coder-logo-light.png) for dark mode automatically
- Dark mode logo variant (coder-logo-dark.png) for light mode
- Updated installer icons and app icons across all platforms

## Other Changes

- Regenerated all icon assets from new source logo (1000x1000 PNG)
- SVG wrappers for renderer brand assets (coder-logo.svg, coder-mark.svg)

---

## Install

**Windows**: Download `Coder-Desktop-0.1.1-setup-win-x64.exe` and run it. The installer supports automatic updates.

**macOS (Intel)**: Download `Coder-Desktop-0.1.1-mac-x64.zip`, extract, and move `Coder Desktop.app` to Applications.

**macOS (Apple Silicon)**: Download `Coder-Desktop-0.1.1-mac-arm64.zip`, extract, and move `Coder Desktop.app` to Applications.

All artifacts include SHA256 checksums in the release manifest.

---

## Notes

- macOS application ZIPs must be built on macOS or by the macOS GitHub Actions workflow.
- Windows SmartScreen may warn on unsigned builds; a trusted code-signing certificate is recommended for production.
- This is an early release — expect rough edges and rapid iteration.
