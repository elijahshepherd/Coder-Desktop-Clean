# Coder Desktop - Version 0.0.36

**Release date:** June 7, 2026

**Version:** `0.0.36`

**Status:** Patch

## Overview

Coder Desktop 0.0.36 focuses on the annotated UI polish pass for the chat empty state, sidebar logo, Settings personalization controls, and Local settings actions.

This release makes the Coder Desktop logo easier to see in light and dark mode, gives Personalization a richer control layout with calmer motion, cleans up accent color selection, and restores action icons in Local settings.

## What changed

- Increased the visible Coder Desktop mark in the sidebar and empty chat state.
- Kept the dark logo on light mode and the light logo on dark mode through the existing theme switch.
- Reworked the Personalization tab into a more deliberate layout for theme, button tone, custom accent color, and completion feedback.
- Replaced the oversized color-wheel presentation with a smaller custom color control.
- Added subtle animated selection feedback for theme and tone controls.
- Restored icons on Local settings action buttons.
- Checked the updated Settings UI in desktop, dark mode, and mobile layouts.

## Verification

This release was checked with:

- TypeScript type checking.
- Screenshot-driven UI polish checks in desktop and mobile viewports.
- Empty chat and sidebar logo size checks.
- Personalization layout checks in light and dark mode.
- Local settings icon and wrapping checks.
- Horizontal overflow checks for the updated Settings layouts.
