# Coder Desktop - Version 0.0.34

**Release date:** June 7, 2026

**Version:** `0.0.34`

**Status:** Patch

## Overview

Coder Desktop 0.0.34 focuses on settings cleanup, transparent logo assets, and generated-image copy reliability.

This release removes the event-themed personalization option, simplifies the Settings experience, makes the light and dark logo assets background-free, fixes generated image copy links, and removes the missing-duration placeholder from chat feedback actions.

## What changed

- Removed the event-themed personalization option from settings, stored personalization state, safe settings-tool requests, current docs, and changelog wording.
- Removed icons from Settings tabs, headings, provider rows, theme controls, local actions, and security settings.
- Increased Settings text size and cleaned up spacing, selected tab states, dark-mode contrast, and mobile tab layout.
- Rebuilt the light and dark Coder Desktop logo assets with transparent backgrounds.
- Standardized Copy buttons so chat and generated-image copy actions share the same visible behavior.
- Generated image Copy link now copies a short working object URL when the image was stored as a data URL.
- Chat feedback actions no longer show a missing-duration placeholder when a message has no recorded duration.
