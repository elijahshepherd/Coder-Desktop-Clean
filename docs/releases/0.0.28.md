# Coder Desktop - Version 0.0.28

**Release date:** June 6, 2026

**Version:** `0.0.28`

**Status:** Patch

## Overview

Coder Desktop 0.0.28 focuses on settings organization, prompt privacy false-positive handling, stronger workspace boundaries, and more reliable stop behavior.

This release makes settings easier to scan by moving provider, access, personalization, AI functionality, and local maintenance controls into a full-screen tabbed surface. It also adds user-controlled completion feedback, including a subtle finished-card animation and desktop notifications.

## What Changed

- Settings now open as a full-screen tabbed panel instead of one long settings stack.
- Personalization includes button tone choices, completion animation, and desktop notification controls.
- AI functionality includes Max Let Me Knows, disabled by default, for users who want the assistant to ask clarification cards more often instead of inferring details.
- Image generation now keeps only one active image model selection across providers. The selected image model belongs to the primary provider.
- Prompt privacy handling now uses a hidden review hint instead of an immediate hard refusal, reducing false positives on normal questions.
- Stop behavior now ignores late provider and tool results after the user stops a chat.
- File and shell tools now require a selected workspace and reject paths outside that workspace.

## Security Model

The release keeps renderer code behind the preload bridge. New settings and notification behavior are exposed through narrow IPC methods only.

File reads, file writes, folder changes, and shell commands are now stricter about workspace scope. Absolute paths are allowed only when they resolve inside the selected workspace. If a user wants work done somewhere else, that folder should be selected as the workspace first.

Provider API keys remain stored outside public renderer state.

## User Notes

The Ask for approval access description now says: "Always ask before creating edits or using the internet."

When a save location, workspace choice, provider choice, or missing detail blocks a tool action, Coder Desktop now guides the assistant toward a Let Me Know card instead of a plain text pause.
