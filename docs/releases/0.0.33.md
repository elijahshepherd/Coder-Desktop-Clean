# Coder Desktop - Version 0.0.33

**Release date:** June 7, 2026

**Version:** `0.0.33`

**Status:** Patch

## Overview

Coder Desktop 0.0.33 is a small follow-up to the 0.0.32 update flow.

This release changes the visible update prompt so it shows the new version and one direct `Click here to install` action. That action opens the matching GitHub Release download for the user's operating system and CPU architecture.

## What changed

- The corner update card now says there is a new update.
- The card shows the new version number.
- The card has one `Click here to install` button instead of the previous two-button layout.
- The settings update action uses the same `Click here to install` wording.
- The app still opens the trusted GitHub Release asset instead of installing directly or closing the app.

## Verification

This release was checked with:

- TypeScript type checking.
- Full Vitest suite.
- Production build verification.
- Screenshot-driven update-card inspection in desktop and mobile browser viewports.

## Downloads

Versioned downloads are stored under `downloads/0.0.33` and published through GitHub Releases. Windows artifacts can be built locally on Windows. macOS ZIP artifacts are produced by the macOS release workflow and uploaded to the matching GitHub Release.

Users should download application binaries from the official GitHub Release for `v0.0.33`.
