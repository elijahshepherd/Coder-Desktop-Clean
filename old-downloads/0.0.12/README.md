# Coder Desktop 0.0.12 Downloads

This folder contains the local Windows release artifacts for Coder Desktop `0.0.12`.

> [!IMPORTANT]
> Users should download from the matching GitHub Release when possible because the release page contains the published binary assets and checksums.

Built locally on Windows:

- `Coder-Desktop-0.0.12-setup-win-x64.exe`
- `Coder-Desktop-0.0.12-setup-win-x64.exe.blockmap`
- `Coder-Desktop-0.0.12-win-x64.zip`
- `Coder-Desktop-0.0.12-win-arm64.zip`
- `manifest.json`

Published by GitHub Actions on the release page:

- `Coder-Desktop-0.0.12-mac-x64.zip`
- `Coder-Desktop-0.0.12-mac-arm64.zip`

The Windows setup installer and ZIP archives are produced from the packaged Electron desktop app. The macOS ZIP files are produced by the macOS GitHub Actions runner because macOS applications must be packaged on macOS.
