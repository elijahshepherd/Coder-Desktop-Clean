# Coder Desktop 0.0.1 Downloads

Real release artifacts for version `0.0.1` live in this folder.

## Included locally

- `Coder-Desktop-0.0.1-win.exe`
- `Coder-Desktop-0.0.1-win-x64.exe`
- `Coder-Desktop-0.0.1-win-x64.zip`
- `Coder-Desktop-0.0.1-win-arm64.exe`
- `Coder-Desktop-0.0.1-win-arm64.zip`

## macOS

macOS application ZIPs must be built on macOS. The repository includes `.github/workflows/build-macos-downloads.yml`, which builds:

- `Coder-Desktop-0.0.1-mac-x64.zip`
- `Coder-Desktop-0.0.1-mac-arm64.zip`

The workflow commits those files back into this folder with Git LFS.

## Manifest

`manifest.json` records artifact sizes and SHA-256 hashes.
