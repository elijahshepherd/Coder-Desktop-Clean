# Coder Desktop 0.0.4 Downloads

This folder contains the release artifacts for Coder Desktop `0.0.4`.

These files are intended to match the assets published on the GitHub Release page for `v0.0.4`.

## Included Artifacts

- `Coder-Desktop-0.0.4-win-x64.exe`
- `Coder-Desktop-0.0.4-win-x64.zip`
- `Coder-Desktop-0.0.4-win-arm64.exe`
- `Coder-Desktop-0.0.4-win-arm64.zip`
- `Coder-Desktop-0.0.4-mac-x64.zip`
- `Coder-Desktop-0.0.4-mac-arm64.zip`
- `manifest.json`

## Choosing A Download

Use the artifact that matches your device:

- Windows x64: use `Coder-Desktop-0.0.4-win-x64.exe`.
- Windows ARM64: use `Coder-Desktop-0.0.4-win-arm64.exe`.
- Windows ZIP users: choose the matching `win-x64.zip` or `win-arm64.zip`.
- macOS Intel: use `Coder-Desktop-0.0.4-mac-x64.zip`.
- macOS Apple Silicon: use `Coder-Desktop-0.0.4-mac-arm64.zip`.

## Manifest

`manifest.json` records artifact sizes and SHA-256 hashes after artifacts are built.

Use the manifest to confirm that a downloaded artifact matches the expected size and hash.

## Notes

Windows artifacts are built locally on Windows. macOS artifacts are built on macOS through the GitHub Actions workflow.

If a local checkout shows macOS ZIP files as very small pointer files, use the GitHub Release page for the real macOS downloads.
