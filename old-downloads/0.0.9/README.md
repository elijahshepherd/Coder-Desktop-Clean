# Coder Desktop 0.0.9 Downloads

This folder contains release artifacts for Coder Desktop `0.0.9`.

These files are intended to match the assets published on the GitHub Release page for `v0.0.9`.

## Included Artifacts

- `Coder-Desktop-0.0.9-setup-win-x64.exe`
- `Coder-Desktop-0.0.9-setup-win-x64.exe.blockmap`
- `Coder-Desktop-0.0.9-win-x64.zip`
- `Coder-Desktop-0.0.9-win-arm64.zip`
- `Coder-Desktop-0.0.9-mac-x64.zip`
- `Coder-Desktop-0.0.9-mac-arm64.zip`
- `manifest.json`

macOS ZIP artifacts are built by the GitHub Actions macOS runner and published through GitHub Releases.

## Choosing A Download

- Windows setup installer: use `Coder-Desktop-0.0.9-setup-win-x64.exe`.
- Windows ZIP archive: use `Coder-Desktop-0.0.9-win-x64.zip`.
- Windows ARM64 ZIP archive: use `Coder-Desktop-0.0.9-win-arm64.zip`.
- macOS Intel ZIP archive: use `Coder-Desktop-0.0.9-mac-x64.zip`.
- macOS Apple silicon ZIP archive: use `Coder-Desktop-0.0.9-mac-arm64.zip`.

## Manifest

`manifest.json` records artifact sizes and SHA-256 hashes after artifacts are built.

Use the manifest to confirm that a downloaded artifact matches the expected size and hash.

## Notes

The Windows setup installer and ZIP archive are real packaged desktop application downloads. The setup installer is the preferred Windows download for users who want a normal installed desktop app.
