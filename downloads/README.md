# Downloads

This folder contains release artifacts for Coder Desktop versions.

## Structure

Each version has its own folder (e.g., `0.0.43/`) containing:

- `Coder-Desktop-{version}-setup-win-x64.exe` - Windows installer
- `Coder-Desktop-{version}-win-x64.exe` - Windows portable executable
- `Coder-Desktop-{version}-win-x64.zip` - Windows ZIP archive
- `Coder-Desktop-{version}-mac-x64.zip` - macOS x64 ZIP (when available)
- `Coder-Desktop-{version}-mac-arm64.zip` - macOS ARM64 ZIP (when available)
- `manifest.json` - Download manifest for the app

## Official Downloads

Official downloads should come from the [GitHub Releases](https://github.com/elijahshepherd/Coder-Desktop/releases) page. The files in this folder mirror the release artifacts for the repository archive.
